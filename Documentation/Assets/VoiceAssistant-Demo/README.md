# VoiceAssistant Demo Media

Current status: `published`

This directory is the canonical home for real showcase media for the `VoiceAssistant` path.

## Allowed Asset Files

- `voiceassistant-success.png`
- `voiceassistant-run.mp4`
- `caption.txt`

## Publish Rule

Only switch status to `published` after all allowed asset files exist and show real success signals from the maintained demo path.

## Required Visual Content

- screenshot must show `Intent`, `Detected language`, `Summary`, `Redacted`, and the final assistant response together
- recording must start from app launch and end after `Process` succeeds
- caption should describe the assistant-style response path without claiming agent-runtime leadership

## Truthfulness Rules

- do not market this as flagship media; this remains a published secondary-demo surface, not the repo's flagship proof path
- do not present it as proof of speech recognition or autonomous agent leadership
- media can support distribution, but it cannot replace benchmark or release proof
