# Flagship Demo Media

Current status: `published`

This directory is the canonical home for real showcase media for the `Intelligent Camera` flagship path.

## Allowed Asset Files

- `intelligent-camera-success.png`
- `intelligent-camera-run.mp4`
- `caption.txt`

## Publish Rule

Only switch status to `published` after all allowed asset files exist and show real success signals from the maintained demo path.

## Required Visual Content

- screenshot must show `Top labels`, `OCR`, `Summary`, and `Privacy preview` together
- recording must start from app launch and end after `Analyze Frame` succeeds
- caption should use the baseline line from the flagship demo pack unless a better truthful caption replaces it

## Truthfulness Rules

- do not add simulator output as if it were mobile release proof
- do not crop out missing or empty success signals
- do not publish edited media that changes the real app result
- media can support distribution, but it cannot replace benchmark or release proof
